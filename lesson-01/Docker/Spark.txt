# Демонстрация Spark

## Запускаем Spark
$ docker compose up -d
$ docker compose ps -a

## Веб-интерфейс:
- Spark Master http://localhost:8080
- Jupyter http://localhost:8888

## Находим URL с токеном для подключения к Jupyter
$ docker exec python jupyter server list

## Копируем URL, меняем python на localhost и открываем его.
